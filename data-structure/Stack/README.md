# Stack

> 스택 (Stack)
>
> **스택**은 아래의 두가지 연산을 가진 요소들의 집합인 추상 자료형입니다.
>
> - **push**는 집합에 요소를 추가하는 것이며,
> - **pop**은 아직 제거되지 않은 가장 최근에 추가된 요소를 제거하는 연산입니다.
>
> LIFO (last in, first out) 이다.
